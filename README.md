# Chef aws_tools cookbook

Installs all the AWS Command Line Tools

* ec2-cli
* autoscaling cli 
* elastic load balancing cli
* rds cli
* cloudformation cli

# Requirements
* Ruby 1.9+
* Java
* Tested on Ubuntu 10.04
